# BOQ Calculator

Simple standalone web app to calculate a Bill of Quantities (BOQ).

## Features
- Project info (project, client, prepared by, date, currency)
- Itemized BOQ rows (description, unit, qty, rate, waste %)
- Auto calculation for item amount and full summary
- Discount, tax, and contingency support
- Save/load from browser local storage
- Export to CSV and JSON

## Run
Open `index.html` in any modern browser.
